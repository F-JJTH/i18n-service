How to integrate Kizeo Translate to your application

Kizeo translate provide 2 outputs:

  • Languages available for a project
  • Translations available for a project

Available environment names:

  • dev
  • preprod
  • prod

Available language codes: see below «List of languages» section or go to «Parameters» then «Languages»

Environment dev is updated after each modification in «Definition» and «Translation» section

Environment preprod and prod is updated after each deployment in «Deploy» section

Your project ID is: {{ productId }}

List of languages

To get list of languages code available: https://client.translate.kizeo.dev/ project ID/ environment/languages.json
The first entry of this endpoint is the code of the default language set in your project

Translations

To get translations available: https://client.translate.kizeo.dev/ project ID/ environment/ language code.json
List of language code can be found above in «List of languages» section
